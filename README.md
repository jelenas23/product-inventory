**Product Inventory**

A Spring Boot service for managing product inventory with basic functionality to save/update products and retrieve product quantities.
This project demonstrates a small inventory service built using Spring Boot and Spring Data JPA. It includes a service layer that interacts with a repository, and unit tests to ensure correctness.

**Features:**
Save or update product inventory,
retrieve aggregated product quantities via DTOs

**Tech Stack:** Java, Spring Boot, Spring Data JPA, JUnit, Mockito
